The info page

What is "The Typing Site" and why I made it

- The Typing Site is a website with games that are intended to help people improve their typing skills, while still being fun

- I decided to make this project partially because I wanted to improve my CSS, HTML, and JavaScript skills, but also because I saw a lot of potential in making a website where I can add games indefinitely

Project development

- I started this project as a complete newbie to JavaScript and without ever having made a single other website, so it was quite the struggle at first. But thanks to AI and all the leaning materials available nowadays it wasn't impossible

- At first I didn't even intend to make a site with multiple games, I just wanted to play around and had the idea of using player imput to do something. I learned how to make an event listener and set it to keydown, then I made it so that every key a player presses gets displayed on the screen. Then I had the idea to make a game out of it, random words get displayed on the screen and you have to type them out. That's how the Tower game was created.

- After the first version of my first game I decided to make a lobby, because I had an idea for another one - the Flow game, which was back then called the Combo game and I wanted the players to be able to switch between the games smoothly

- Sadly, I only make a simple version of the lobby without even using JavaScript and then I gave up on the project. That was until we got assigned a midterm project in my web programming class, I never even thought of starting something new, this was it, I finally had motivation to go back to working on my typing games

- I knew that if I wanted the lobby to look any good, I needed more games, so I started two new ideas - using the same system as in the Tower game, but this time the words are scrambled aka. The Scrambles game, for the other idea I wanted to make something more game-like - so I settled on the reaction game, where different keys are shown on the screen and you have to press them.

The future of this project

- I definitely don't want to stop working on this site once I submit it as a school project. Most of the games are just barely playable and not too much fun yet, so I am planning on updating them. Each game will have a starter page with an explanation of what to do in the game and options to change difficulty with individual settings for each game

- Over time I also want to add more games to make this a proper gaming website.
